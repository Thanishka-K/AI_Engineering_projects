# Project 10: Stock Price Trend Predictor

## Overview
A moderate-level Time Series forecasting utility that builds rolling lag matrices to predict upcoming asset trend lines using standard local regression pipelines.

## Technical Details
- **Core Algorithm**: Linear Regression (`LinearRegression`)
- **Feature Strategy**: 1-Day Temporal Lag Shift Processing
- **Data Visuals**: Matplotlib Time-Series Projection Line Mapping
