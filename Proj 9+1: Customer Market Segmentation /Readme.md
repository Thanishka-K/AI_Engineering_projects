# Project 10: Customer Market Segmentation (K-Means Clustering)

## Overview
A moderate-level Unsupervised Machine Learning utility that clusters customer profiles based on annual income and spending metrics to discover underlying behavioral patterns.

## Technical Details
- **Algorithm**: K-Means Clustering (`KMeans`)
- **Category**: Unsupervised Learning (Pattern Detection)
- **Visualization**: Matplotlib scatter plotting with centroid tracking math vectors.
