# Project 5: Plant Disease Detection

## Overview
A Deep Learning application using Convolutional Neural Networks (CNN) to classify plant diseases from leaf images.

## Technical Stack
- **Framework**: TensorFlow/Keras
- **Architecture**: CNN (Convolution -> Pooling -> Dense)
- **Deployment**: Streamlit

## Key Learning
- Image preprocessing (resizing, normalization).
- Feature extraction using filters in CNNs.
- Handling image data formats in Python.
